======================================================
Word Guessing Game - Project
======================================================

Description:
A desktop Word Guessing Game built with Python and Tkinter. The user has six attempts to guess a randomly selected 6-letter word, receiving color-coded feedback for each guess.

Requirements:
- Python 3.x installed on your system.
- Tkinter (This is included in the standard Python library, so no extra installation via pip is required).

How to Run the Application:
1. Ensure all Python source files are extracted to a single folder.
2. Open your terminal or command prompt.
3. Navigate to the folder containing the application using the 'cd' command (e.g., cd Desktop/WordGame).
4. Run the script by typing: python word_game.py
   (Note: If your system uses 'python3', type: python3 word_game.py)

How to Play:
- Type a 6-letter word into the input box.
- Press the 'Enter' key on your keyboard or click 'Submit Guess'.
- GREEN letters indicate the correct letter in the correct spot.
- YELLOW letters indicate the correct letter in the wrong spot.
- GREY letters indicate the letter is not in the word.
- Guess the word in 6 tries to win. Click 'Play Again' to start a new round!
